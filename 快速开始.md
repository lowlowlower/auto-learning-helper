## ⚡ 3分钟快速开始

### 步骤1️⃣：安装插件（1分钟）

1. 打开浏览器，访问：
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`

2. 开启右上角的 **"开发者模式"**

3. 点击 **"加载已解压的扩展程序"**

4. 选择插件文件夹：`d:\github\插件\`

5. 看到"自动学习助手"图标出现在工具栏 ✅

### 步骤2️⃣：测试是否识别课程（1分钟）

1. **打开您的学习平台课程列表页**

2. **按F12打开开发者工具** → 切换到 **Console** 标签

3. **复制下面的代码，粘贴到Console并回车：**

```javascript
// 快速测试代码
const cards = document.querySelectorAll('.item.hover-shadow');
console.log(`%c找到 ${cards.length} 个课程`, 'color: green; font-size: 16px; font-weight: bold');

if (cards.length > 0) {
  cards.forEach((card, i) => {
    card.style.outline = '3px solid red';
    const label = document.createElement('div');
    label.textContent = i + 1;
    label.style = 'position:absolute;top:10px;left:10px;background:red;color:white;width:30px;height:30px;border-radius:50%;display:flex;align-items:center;justify-content:center;font-weight:bold;z-index:9999';
    card.style.position = 'relative';
    card.appendChild(label);
  });
  console.log('%c✅ 插件可以识别这些课程！', 'color: green; font-size: 14px; font-weight: bold');
} else {
  console.log('%c❌ 未找到课程，请提供页面信息', 'color: red; font-size: 14px; font-weight: bold');
}
```

4. **查看结果：**
   - ✅ **成功**：页面上的课程卡片被红色边框标记，并显示序号
   - ❌ **失败**：没有任何高亮，Console显示"找到 0 个课程"

### 步骤3️⃣：开始自动学习（1分钟）

**如果测试成功：**

1. **点击浏览器工具栏的插件图标** 📚

2. **查看设置（可选）：**
   - 视频倍速：1.5x（推荐）
   - ✓ 自动下一课
   - ✓ 循环学习

3. **点击"开始学习"按钮** 🎯

4. **观察运行：**
   - 插件会自动点击第一个课程
   - 跳转到视频页面
   - 自动播放视频（1.5倍速）
   - 完成后返回列表
   - 继续下一个课程

5. **查看进度：**
   - 插件面板显示已学习课程数
   - 底部显示运行日志

## ❓ 常见问题快速解决

### Q1: 测试代码显示"找到 0 个课程"

**原因：** 页面结构可能不同

**解决：**
1. 打开 `调试工具.html`
2. 使用"检测课程卡片"功能
3. 查看详细检测结果
4. 如果仍然不行，提供页面HTML代码

### Q2: 能识别课程但点击没反应

**原因：** 卡片本身可能不可点击

**测试：** 在Console运行
```javascript
document.querySelector('.item.hover-shadow').click();
```

**如果没反应：** 尝试点击内部元素
```javascript
document.querySelector('.item.hover-shadow .cover').click();
```

### Q3: 视频不自动播放

**原因：** 浏览器自动播放策略

**解决：** 
1. 手动点击播放一次
2. 之后会自动播放

### Q4: 播放完成不返回列表

**检查：** 页面是否可以通过浏览器后退按钮返回

**测试：** 在视频页面手动点击后退，看是否回到列表

## 💡 使用技巧

### 提高学习效率

1. **设置2.0x倍速** - 快速刷课
2. **开启循环学习** - 反复巩固
3. **最小化浏览器** - 后台运行
4. **多开标签页** - 同时学习多门课（不推荐，可能被检测）

### 避免问题

1. **保持网络稳定** - 避免视频加载失败
2. **不要关闭标签页** - 可以最小化但不要关闭
3. **定期查看日志** - 及时发现问题
4. **首次使用先测试** - 确保流程正常

### 合理使用

1. **配合真实学习** - 插件仅辅助，不能替代学习
2. **重要内容做笔记** - 关键知识点要记录
3. **适度使用倍速** - 过快可能影响理解
4. **遵守平台规则** - 确保符合学习要求
