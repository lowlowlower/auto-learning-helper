图标文件夹
==========

此文件夹用于存放插件图标文件。

需要的文件：
-----------
icon16.png  - 16x16 像素的PNG图标
icon48.png  - 48x48 像素的PNG图标
icon128.png - 128x128 像素的PNG图标

如何获取图标：
-------------

方法1（推荐）：使用图标生成器
1. 在浏览器中打开项目根目录的 "生成图标.html" 文件
2. 点击"下载全部图标"按钮
3. 将下载的3个PNG文件放入此文件夹

方法2：在线制作
1. 访问 https://favicon.io/favicon-generator/
2. 创建自定义图标
3. 下载并重命名为 icon16.png, icon48.png, icon128.png
4. 放入此文件夹

方法3：使用任意图片
1. 准备一张合适的图片
2. 使用图片编辑软件或在线工具调整为3个尺寸
3. 保存为PNG格式并重命名
4. 放入此文件夹

临时方案：
---------
如果暂时没有合适的图标，可以使用任意PNG图片，
只要文件名和尺寸正确，插件就可以正常加载。

建议的图标主题：
--------------
📚 书本
🎓 毕业帽
✏️ 铅笔
💡 灯泡
📖 打开的书
🏫 学校

完成后：
-------
确保此文件夹包含：
✓ icon16.png
✓ icon48.png
✓ icon128.png

然后重新加载插件即可看到图标显示在浏览器工具栏中。
