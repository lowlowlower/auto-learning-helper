# 📚 自动学习助手 - 浏览器插件

一个强大的浏览器扩展，可以自动播放学习平台的课程，实现持续学习无需人工干预。

## ✨ 主要功能

- 🎯 **自动播放课程** - 自动识别并点击课程卡片
- 🔄 **智能跳转** - 自动处理页面跳转和视频播放
- ⚡ **倍速播放** - 支持1.0x - 2.0x倍速设置
- 📊 **学习统计** - 记录已学习课程数量和进度
- 🔁 **循环学习** - 支持学完所有课程后自动重新开始
- 💾 **进度保存** - 自动保存学习进度，关闭浏览器也不丢失
- 📝 **运行日志** - 实时显示运行状态和操作日志

## 📦 安装方法

### 方法一：开发者模式加载（推荐）

1. **下载插件文件**
   - 确保所有插件文件都在同一个文件夹中

2. **打开浏览器扩展管理页面**
   - Chrome: 访问 `chrome://extensions/`
   - Edge: 访问 `edge://extensions/`

3. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关并打开

4. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择插件所在的文件夹
   - 插件加载成功！

### 方法二：打包安装

1. 在扩展管理页面点击"打包扩展程序"
2. 选择插件目录
3. 生成 .crx 文件
4. 拖拽 .crx 文件到扩展管理页面安装

## 🎮 使用方法

### 基本使用

1. **打开学习平台**
   - 访问您的在线学习平台网站

2. **打开插件**
   - 点击浏览器工具栏中的插件图标

3. **配置设置**
   - 设置视频倍速（默认1.5x）
   - 开启/关闭自动下一课
   - 开启/关闭循环学习

4. **开始学习**
   - 点击"开始学习"按钮
   - 插件会自动开始播放课程

5. **查看进度**
   - 在插件面板中查看已学习课程数量
   - 查看当前正在学习的课程
   - 查看运行日志

### 高级功能

#### 倍速播放
- 支持 1.0x、1.25x、1.5x、1.75x、2.0x 倍速
- 实时调整，立即生效

#### 自动下一课
- 开启后会在课程完成后自动跳转到下一个课程
- 关闭后需要手动点击下一课

#### 循环学习
- 开启后会在所有课程学完后自动重新开始
- 适合需要反复学习的场景

#### 重置进度
- 点击"重置进度"按钮可清除所有学习记录
- 重新开始学习所有课程

## ⚙️ 技术说明

### 文件结构

```
插件/
├── manifest.json          # 插件配置文件
├── background.js          # 后台服务脚本
├── content.js            # 内容脚本（核心逻辑）
├── popup.html            # 弹出界面HTML
├── popup.js              # 弹出界面逻辑
├── popup.css             # 弹出界面样式
├── icons/                # 图标文件夹
│   ├── icon16.png       # 16x16 图标
│   ├── icon48.png       # 48x48 图标
│   └── icon128.png      # 128x128 图标
└── README.md            # 说明文档
```

### 工作原理

1. **页面检测**
   - Content Script 在页面加载后自动运行
   - 智能识别课程列表页和视频播放页

2. **自动化流程**
   ```
   课程列表页 → 点击课程卡片 → 视频播放页 → 设置倍速 → 等待播放完成 → 返回列表 → 下一个课程
   ```

3. **状态管理**
   - 使用 Chrome Storage API 保存所有状态
   - 支持跨页面、跨会话数据持久化

4. **消息通信**
   - Popup ↔ Background ↔ Content Script 三方通信
   - 实时同步状态和设置

### 兼容性

- ✅ Chrome 88+
- ✅ Edge 88+
- ✅ 其他 Chromium 内核浏览器

## 🔧 自定义配置

### 适配不同的学习平台

如果插件无法识别您的学习平台，可以修改 `content.js` 中的选择器：

```javascript
// 在 detectCourseCards() 函数中添加您平台的课程卡片选择器
const selectors = [
  '.your-course-card-class',  // 添加您的选择器
  '.course-card',
  // ...
];
```

### 调整检查间隔

```javascript
// 在 start() 函数中修改检查间隔（毫秒）
checkInterval = setInterval(() => {
  detectPageAndRun();
}, 3000);  // 修改这个值
```

## 📝 注意事项

1. **网络连接**
   - 确保网络连接稳定，避免视频加载失败

2. **页面停留**
   - 学习期间请保持浏览器窗口打开
   - 可以最小化窗口，但不要关闭标签页

3. **学习时长**
   - 插件会等待视频真实播放完成
   - 不会跳过学习时长要求

4. **平台兼容**
   - 首次使用建议先测试一个课程
   - 确认自动化流程正常工作

5. **合理使用**
   - 本插件仅用于辅助学习
   - 建议配合真实学习使用

## 🐛 常见问题

### Q1: 插件无法识别课程
**A:** 可能是页面结构不匹配，需要自定义选择器。查看浏览器控制台的日志信息。

### Q2: 视频不自动播放
**A:** 某些浏览器有自动播放策略限制，需要先手动播放一次视频。

### Q3: 无法返回课程列表
**A:** 检查页面是否有返回按钮，或者等待浏览器自动后退。

### Q4: 学习进度丢失
**A:** 确保没有清除浏览器数据，插件使用 localStorage 保存进度。

### Q5: 插件图标不显示
**A:** 需要在 icons 文件夹中放置图标文件（见下方说明）。

## 🎨 图标文件说明

插件需要三个尺寸的图标文件，请创建 `icons` 文件夹并添加：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

### 快速生成图标

您可以：
1. 使用在线图标生成工具（如 favicon.io）
2. 使用 📚 表情符号生成图标
3. 使用任何图片编辑软件创建

或者临时使用纯色图片：

```bash
# 在插件目录下执行（需要 ImageMagick）
mkdir icons
convert -size 16x16 xc:#667eea icons/icon16.png
convert -size 48x48 xc:#667eea icons/icon48.png
convert -size 128x128 xc:#667eea icons/icon128.png
```
