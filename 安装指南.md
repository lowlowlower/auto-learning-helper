# 🚀 自动学习助手 - 快速安装指南

## 📋 安装前准备

### 1. 准备图标文件

插件需要3个图标文件才能正常显示。请选择以下任一方法：

#### 方法A：使用图标生成器（最简单）✨

1. 用浏览器打开 `生成图标.html` 文件
2. 点击 **"下载全部图标"** 按钮
3. 将下载的3个文件（icon16.png、icon48.png、icon128.png）放入 `icons` 文件夹

#### 方法B：在线制作

访问 [https://favicon.io/favicon-generator/](https://favicon.io/favicon-generator/)
- 选择文字或Emoji：📚 或 "学"
- 背景色选择：#667eea
- 下载后重命名并放入 `icons` 文件夹

#### 方法C：临时方案

如果只是测试，可以使用任意3张PNG图片，重命名为：
- icon16.png
- icon48.png  
- icon128.png

放入 `icons` 文件夹即可（图标不好看但不影响功能）

## 💻 安装插件

### Chrome / Edge 浏览器

#### 步骤1：打开扩展管理页面

**Chrome:**
- 地址栏输入：`chrome://extensions/`
- 或：菜单 → 更多工具 → 扩展程序

**Edge:**
- 地址栏输入：`edge://extensions/`
- 或：菜单 → 扩展 → 管理扩展

#### 步骤2：启用开发者模式

在页面右上角找到 **"开发者模式"** 开关，确保它是 **开启** 状态

#### 步骤3：加载插件

1. 点击 **"加载已解压的扩展程序"** 按钮
2. 在弹出的文件选择器中，选择插件所在的文件夹
   ```
   D:\github\插件\
   ```
3. 点击"选择文件夹"

#### 步骤4：确认安装成功

- 插件列表中出现 **"自动学习助手"**
- 浏览器工具栏显示插件图标
- 状态显示为 **已启用**

## 🎯 首次使用

### 1. 打开学习平台

访问您的在线学习网站（如截图中的学习平台）

### 2. 打开插件面板

点击浏览器工具栏中的插件图标

### 3. 配置设置（可选）

- **视频倍速**：推荐1.5x（可根据需要调整）
- **自动下一课**：勾选（完成后自动进入下一个课程）
- **循环学习**：勾选（学完所有课程后重新开始）

### 4. 开始学习

点击 **"开始学习"** 按钮，插件会自动：
1. 识别页面上的课程卡片
2. 点击第一个未学习的课程
3. 自动播放视频（设定的倍速）
4. 视频结束后返回列表
5. 继续下一个课程

### 5. 查看进度

在插件面板中可以看到：
- ✅ 运行状态：运行中 / 已停止
- 📊 已学习课程数量
- 📖 当前正在学习的课程
- 📝 运行日志

## ⚙️ 常用操作

### 暂停学习
点击 **"停止学习"** 按钮（进度会保存）

### 继续学习
再次点击 **"开始学习"** 按钮（从上次位置继续）

### 重置进度
点击 **"重置进度"** 按钮，清除所有学习记录

### 修改倍速
在插件面板中选择新的倍速，立即生效

## 🔍 故障排除

### 问题1：图标不显示
**原因**：icons文件夹中缺少图标文件  
**解决**：按照上方"准备图标文件"步骤操作

### 问题2：插件无反应
**原因**：页面未加载完成，或插件未正确注入  
**解决**：刷新页面后重试

### 问题3：无法识别课程
**原因**：学习平台页面结构与插件不匹配  
**解决**：
1. 打开浏览器开发者工具（F12）
2. 查看Console中的日志
3. 可能需要自定义选择器（见README.md高级配置）

### 问题4：视频不自动播放
**原因**：浏览器自动播放策略限制  
**解决**：手动点击播放一次，后续会自动播放

### 问题5：学习进度丢失
**原因**：清除了浏览器数据  
**解决**：避免清除浏览器存储数据（localStorage）
